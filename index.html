Part 2

Aim: to remove an object against a complex background in Adobe Photoshop. Content-Aware Fill analyzes the image to find the best detail to intelligently replace a selected area, and it gives you controls for fine-tuning the result.

Steps:

  1. select the object using Object Selection tool
  2. with the selection active, go to Edit menu and choose Content Aware-Fill
  3. choose "Auto", "Rectangular" or "Custom" option from the Sampling Area options
  4. use any sampling Brush tool with any of the above-mentioned options to modify the area that is being sampled
  5. choose appropriate Fill and Output settings and finally, Apply changes
